# vps_setup-cookbook

Sets up a basic vps on linode or digital ocean.

## Usage

### vps_setup::default

Include `vps_setup` in your node's `run_list`:

```json
{
  "run_list": [
    "recipe[vps_setup::default]"
  ]
}
```
